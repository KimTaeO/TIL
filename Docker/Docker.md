## 도커란
* 도커는 리눅스 컨테이너에 리눅스 어플리캐이션 프로세스 격리기술을 사용하여 더 쉽게 컨테이너를 실행하고 관리할 수 있도록 하는 가상화 플랫폼이다
* 독립적인 환경을 구성해 하드웨어를 효율적으로 사용하는 기술
    * ```가상화```는 소프트웨어를 활용해 하드웨어의 스토리지, 프로세서, 메모리를 ```가상 머신```이라는 다수의 가상 컴퓨터로 분할하는 기술이며, 다수의 가상 컴퓨터는 서로 독립적으로 작동한다
    * ```컨테이너```는 어떤 환경에서도 애플리케이션 코드를 컨테이너를 설치해 구동함으로써 어플리케이션을 별도의 설정 없이 쉽게 사용할 수 있도록 하는 기술이다
